About This Project

Project Overview

This project is an AI-powered web application designed to detect different types of brain tumors from MRI scans using deep learning. The system analyzes uploaded MRI images and predicts whether a tumor is present, along with the tumor type.

How It Works

The uploaded MRI scan is processed by a Convolutional Neural Network (CNN) model trained on brain tumor datasets. The model predicts tumor categories such as Glioma, Meningioma, Pituitary tumor, or No tumor. A heatmap visualization highlights the regions of the image that influenced the model’s decision.

Features

• Brain tumor classification using deep learning
• AI explanation for predictions
• Lesion heatmap visualization
• Prediction confidence score
• Downloadable medical-style report

Technology Stack

Python Flask TensorFlow / Keras OpenCV HTML CSS

Purpose

This project demonstrates how Artificial Intelligence can assist in medical image analysis and support doctors in early tumor detection. It is intended for educational and research purposes.
